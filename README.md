# 🏥 펫호 (Petho)
> **이색반려동물 진료 원스톱 서비스** 모바일 앱 (HTML/CSS/JS 단일 파일)

---

## ✅ 구현된 기능

### 📱 화면 구성 (11개 스크린)
| 스크린 ID | 화면명 | 설명 |
|-----------|--------|------|
| `splash` | 스플래시 | 앱 진입, 시작하기·둘러보기 버튼 |
| `home` | 홈 | 검색, 빠른 메뉴(8종), 배너 슬라이더, 가까운 병원 목록 |
| `find` | 병원 찾기 | 지도 뷰 + 필터 칩 + 병원 리스트 |
| `booking` | 진료 예약 | 4단계 예약 플로우 (달력·시간 선택) |
| `mypet` | 내 반려동물 | 건강 대시보드, 예방접종 타임라인 |
| `records` | 진료기록 | 진료 이력, 비용, 처방 내역 |
| `mypage` | 마이페이지 | 프로필, 통계, 메뉴 |
| `detail` | 병원 상세 | 진료 시간, 리뷰, 예약 버튼 |
| `insurance` | **🛡️ 펫보험** | 보험 상품 비교, 보험료 계산기, 가입 신청 |
| `review` | **⭐ 병원 후기** | 후기 목록, 별점 분포, 후기 작성 모달 |
| `funeral` | **🕊️ 장례케어** | 장례 절차 안내, 추모 서비스, 펫로스 상담 |
| `funeralHall` | **🏛️ 장례식장 찾기** | 주변 장례식장 지도, 시설 목록, 전화/예약 |

### 🎨 디자인 시스템
- **Primary**: `#4D9FEC` (메인 블루)
- **Secondary**: `#FF7043` (액센트 오렌지)
- **Funeral Theme**: `#5C6B73` / `#A67C52` (따뜻한 장례 테마)
- **모바일 프레임**: 390×844 iPhone 사이즈
- **반응형**: 440px 이하 모바일 전체화면 대응

## 🔗 기능 진입 경로
| 기능 | 진입 방법 |
|------|-----------|
| 펫보험 | 홈 > 빠른 메뉴 "펫보험" / 마이페이지 > "펫보험 관리" |
| 병원 후기 | 홈 > 빠른 메뉴 "병원후기" / 마이페이지 > "내 후기 관리" / 병원 상세 > "후기" 버튼 |
| 장례케어 | 홈 > 빠른 메뉴 "장례케어" / 마이페이지 > "장례케어 서비스" |
| 장례식장 찾기 | 홈 > 빠른 메뉴 "장례식장" / 장례케어 > "주변 장례식장 찾기" 버튼 |
| 후기 작성 | 병원 후기 화면 > 상단 펜 아이콘 또는 하단 "후기 작성하기" 버튼 |
| 보험료 계산 | 펫보험 화면 > 하단 "보험료 계산기" 섹션 |
