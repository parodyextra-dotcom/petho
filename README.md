# 🐾 펫호 (Petho) v2.0
> **반려동물 진료 원스톱 서비스** — 강아지·고양이·이색동물 모두 케어

---

## ✅ 구현된 기능

### 📱 화면 구성 (9개 스크린)
| 스크린 ID | 화면명 | 설명 |
|-----------|--------|------|
| `splash` | 스플래시 | 앱 진입, 반려동물 종류 소개 |
| `home` | 홈 | 검색, 퀵메뉴, 이색동물 배너, 이벤트 배너, 병원목록 |
| `find` | 병원찾기 | 목업 지도 + 동물별 필터 칩 + 병원리스트 |
| `booking` | 진료예약 | 4단계 예약 플로우 (병원→날짜→시간→확인) |
| `mypet` | 내 반려동물 | 강아지/고양이/이색동물 탭, 건강 대시보드, 접종 타임라인 |
| `records` | 진료기록 | 펫별 진료이력, 비용, 처방 태그 |
| `mypage` | 마이페이지 | 프로필, 통계, 서비스 메뉴 |
| `detail` | 병원상세 | 진료 동물 목록, 후기 목록, 후기 작성 |
| `funeral` | 장례서비스 | 화장/납골/수목장/추모식 가격 + 예약 폼 |
| `shop` | 용품쇼핑몰 | 카테고리 필터, 검색, 상품 그리드, 장바구니 |

---

### 🐾 동물 종류 지원
- **강아지** 🐶 — 일반 진료, 예방접종, 미용
- **고양이** 🐱 — 일반 진료, 중성화, 건강검진
- **이색동물 전문** (강조) 🦜🦎🐰🐠🐹
  - 조류 (앵무새, 유황앵무 등)
  - 파충류 (도마뱀, 거북이 등)
  - 소동물 (토끼, 햄스터, 고슴도치)
  - 어류 (금붕어, 열대어 등)
  - 양서류

---

### ⭐ 후기 시스템
- 병원 상세 페이지에서 후기 목록 확인
- **후기 작성 모달**: 별점(1~5), 반려동물 종류 선택, 텍스트 입력
- 작성 즉시 목록에 반영

---

### 🕊️ 장례 서비스
- **개인 화장**: 소형/중형/대형/이색동물별 가격 표시
- **납골함**: 일반/프리미엄/반영구 납골당
- **수목장/자연장**: 단독·합장·잔디장·해양산골
- **추모식**: 가족추모식, 기념품 제작, 방문 수거
- 예약 폼 (서비스 종류, 반려동물 종류, 날짜, 연락처)

---

### 🛒 용품 쇼핑몰
- **카테고리 필터**: 전체/강아지/고양이/이색동물/사료/장난감/건강용품
- **검색 기능**: 상품명, 브랜드 실시간 검색
- **10개 상품**: SALE·NEW·HOT 뱃지, 할인가 표시
- **장바구니**: 수량 조절, 총액 계산, 결제하기

---

### 🧭 하단 네비게이션 (항상 표시)
- 모든 스크린에서 하단 네비게이션 고정 표시
- 홈 / 병원찾기 / 예약(FAB) / 내펫 / 마이
- 현재 화면 활성화 표시
