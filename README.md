# 🏥 펫호 (Petho) v2.0
> **강아지·고양이·이색반려동물 진료 원스톱 서비스** 모바일 앱 (HTML/CSS/JS 단일 파일)

---

## ✅ 구현된 기능

### 📱 화면 구성 (10개 스크린)
| 스크린 ID | 화면명 | 설명 |
|-----------|--------|------|
| `splash` | 스플래시 | 앱 진입, 로그인·둘러보기 버튼 |
| `home` | 홈 | 이색동물 강조 배너, 빠른 메뉴, 가까운 병원 목록 |
| `find` | 병원 찾기 | 지도 뷰 + 필터 칩 + 병원 리스트 (동물별 필터) |
| `booking` | 진료 예약 | 4단계 예약 플로우 (동물선택·달력·시간·확인) |
| `mypet` | 내 반려동물 | 건강 대시보드, 예방접종 타임라인 |
| `records` | 진료기록 | 진료 이력, 비용, 처방 내역 |
| `mypage` | 마이페이지 | 프로필, 통계, 전체 메뉴 |
| `detail` | 병원 상세 | 진료 시간, 리뷰 목록, **후기 작성** |
| `funeral` | 장례 서비스 | 화장/납골/수목장 가격표 + 예약 폼 |
| `insurance` | 펫 보험 | 강아지·고양이·이색동물 전용 보험 플랜 |

### 🦎 이색동물 강조
- 홈 화면 최상단 이색동물 전용 배너
- 이색동물 전용 뱃지 (주황색 그라데이션)
- 병원 찾기 필터: 파충류 / 조류 / 어류 개별 필터
- 예약 시 이색동물 종류 선택 (`이색` 뱃지 표시)
- 보험: 이색동물 전용 플랜 2종
- 장례: 이색동물 전용 가격 항목

### 🐶🐱 강아지·고양이 포함
- 모든 화면에서 강아지·고양이 동등 지원
- 병원 카드에 강아지/고양이 뱃지
- 보험 기본·프리미엄 플랜 (강아지·고양이 전용)

### ⭐ 후기 작성
- 병원 상세 화면에서 별점 (1~5점) 선택
- 텍스트 후기 입력 및 즉시 반영
- 기존 후기 목록 표시 (병원별)

### 🕊️ 장례 서비스
- **화장**: 소형/중형/대형/초대형/이색동물 가격표
- **납골**: 기간별(1년/3년/영구) 안치 가격표
- **수목장**: 기본/지정목/프리미엄/꽃나무 가격표
- **예약 폼**: 서비스 유형 · 동물 종류 · 날짜/시간 · 픽업 신청

### 🛡️ 펫 보험

- 프리미엄 플랜 (₩35,900/월) - 강아지·고양이
- 이색동물 전용 플랜 (₩22,900/월)

- 동물 종류별 필터 (전체/강아지/고양이/이색동물)

### 🗺️ 하단 네비게이션 (항상 표시)
- **홈** / **병원찾기** / **예약(FAB)** / **내펫** / **마이** 5개 탭
- 스플래시 화면 제외 모든 화면에서 고정 표시
- z-index: 9999으로 항상 최상단 레이어
